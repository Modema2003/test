ДЗ по GIT (к 8 июня)

1) а)   Создать в Main текстовый файл, добавить туда текст
        touch 1.txt
   б) открыть текстовый файл и добавить текст:
        nano 1.txt
        (Добавить текст, сохранить, выйти)

2) а) Создать другую ветку
        git branch feature
  б) перейти на новую ветку
        git checkout feature

3) Сделать изменения в текстовом файле в новой ветке
   а) открыть текстовый файл и добавить текст:
        nano 1.txt
        (Изменить текст, сохранить, выйти)

4) Добавить, закоммитить изменения
       а) git add 1.txt
       б) git commit -m "first commit"

5) Перейти в Main
         git checkout main

6) Вмержить новую ветку в Main
         git merge feature

9) Новую ветку запушить во внешний репозиториий
        git push -u origin feature

10) Обновлённый Main запушить во внешний репозиторий.
        git push -u origin main
